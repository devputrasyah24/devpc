# Dev PC — Proyek Praktikum Pemrograman Web Dasar (HTML)

Proyek ini merupakan tugas **Final Project** untuk praktikum **pemrograman web dasar HTML**. Halaman utama proyek berisi website sederhana bertema toko **Dev PC** yang menyediakan **part komputer original** dan layanan **PC rakitan custom**.

## Isi Website
- **Header & marquee** pengumuman promo
- **Navigasi internal** menggunakan tautan anchor
- **Hero section** (banner utama)
- **Bagian Tentang Perusahaan** (visi, misi, dan deskripsi singkat)
- **Multimedia** (video profil YouTube + musik latar audio HTML5)
- **Katalog produk** (tabel produk dengan tampilan grid)
- **Daftar harga & paket rakitan** (tabel kompleks dengan `rowspan`/`colspan`)
- **Formulir pemesanan & konsultasi** (input teks, radio, checkbox, textarea)
- **Google Maps** dan **informasi lokasi toko**
- **Footer**

## Cara Menjalankan

Karena proyek ini murni HTML, cara menjalankannya:
1. Buka file `index.html` di browser favorit.
2. Pastikan seluruh asset berada di lokasi yang benar: `assets/`.

> Catatan: jika browser memblokir autoplay audio, klik tombol play pada kontrol audio.

## Teknologi yang Digunakan
- HTML
- Elemen dasar HTML: tabel, form, iframe, audio, marquee, dan anchor internal.

## Informasi Lanjutan
Website ini dibuat sebagai media pembelajaran dan menampilkan contoh struktur HTML yang cukup lengkap untuk tingkat **pemrograman web dasar**.
